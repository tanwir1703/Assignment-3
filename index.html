Here Are Some Instructions for this Ludo.

There are 2 tokens for each player. Initially both are in locker.
Click On the die to generate a number (Starting from A).
A token can be freed from the Locker only when die rolls 6. Tokens are freed at START point of the player.
To move a token, simply click on the token you wish to move.
If a player's token moves to a cell already occupied by opponent, the player's token replaces those of the opponent, while the opponent's token(s) return to its locker.
When a token reaches its end-point, it leaves the game board. The Player whose all 2 tokens leave the board first, wins.
